# 1DV607-Workshop2
Welcome!

Instructions for running this software. Pre-requisites: Java. This software needs java installed. This software also needs a java compiler, eclipse is recommended.
This application can be run by using the .jar file that was attached to this download. Simply navigate to the folder through CMD, and use the command java -jar "workshop2.jar"

Note, that if you are unable to save the register, you have to open the CMD as an administrator, and follow the procedure above.


If the .Jar file is not working for you then:
Download and open this project in your java compiler.
Open the "Main.class".
Run program.
Use the console to navigate.
